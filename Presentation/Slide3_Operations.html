Core Operations

Task Management

  • Create tasks with title, description, and priority
  • Edit task details anytime
  • Move tasks between status columns (To-Do ↔ In-Progress ↔ Done)
  • Delete tasks when no longer needed
  • Simple sequential IDs (T1, T2, T3...) for easy reference

Time Tracking

  • Start timer when beginning work on a task
  • Pause timer without losing progress
  • Stop timer to log time spent
  • Automatic time calculation and accumulation
  • Only one timer can run at a time

Task Organization & Search

  • Assign tasks to team members
  • Add multiple tags for categorization
  • Set priority levels (low, medium, high)
  • Search tasks by title or description keywords
  • Filter by priority, tag, or assigned user
